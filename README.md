# FinSight Analytics Platform

Real-time financial intelligence platform that automates stock market data workflows.

## Tech Stack
- **Python** — ETL pipeline, data ingestion
- **PostgreSQL** — database (via Docker)
- **GitHub Actions** — CI/CD pipeline
- **Excel** — dynamic reporting dashboard

## Project Status
- Phase 1: Database schema (4 tables, 50 stocks)
- Phase 2: Python ETL pipeline (12,500 rows of data)
- Phase 3: Advanced SQL analytics
- Phase 4: GitHub CI/CD
- Phase 5: Excel dashboard
